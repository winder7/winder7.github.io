Implementação A manipulação de banco de dados através da linguagem Java é de suma importância para o desenvolvedor de software. São poucas etapas para realização deste procedimento:

Baixar o driver do postgres Digite no google "postgresql java driver", acesse o primeiro link e baixe o driver Adicionar o driver ao projeto Crie um projeto no eclipse Botão direito do mouse sobre o nome do projeto Escolha a opção "Propriedades" Escolha, no lado esquerdo, a opção "Java Build Path" Clique sobre a guia "Libraries" Clique no botão "Add External JAR's" Escolha o driver baixado e pressione OK OK Obs: o procedimento é similar ao NetBeans Incluir os comandos de conexão e execução de instrução na linguagem Estabelecer Conexão: DriverManager e Connection Encaminhar instrução: PreparedStatement Processar Resultados Fechar Conexão:
