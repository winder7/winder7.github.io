Projeto de Redes de Computadores

Um projeto de redes de computadores é um estudo (ou planejamento) detalhado, destinado a implantação de uma rede de computadores que possa satisfazer as necessidades de alguém (pessoa física ou jurídica), sendo o compartilhamento de informações e recursos o objetivo mais comum.

Dependendo da natureza do projeto, ele deverá atender não só as expectativas do presente, mas também estar adequado para as mudanças no futuro, isto é, ser “modular” de forma a permitir o crescimento da rede, conforme o surgimento de novas necessidades, sem prejudicar o atual estado de sua funcionalidade.

Tecnologias de software e hardware de redes

Tecnologia escolhida

Para disponibilizar a o acesso a Loja de Serviços Online desenvolvido na disciplina de programação com frameworks, foi escolhida uma plataforma de hospedagem online Java.

Vantagens de usar hospedagem em vez de um servidor próprio:

  • Garantia de disponibilidade;
  • Rapidez, privacidade e segurança, não necessitando de gastos adicionais com manutenção de hardware;
  • Requisitos de hardware podem ser adquiridos com facilidade conforme o crescimento e a necessidade do site. O investimento não precisa ser alto. Se o seu site ainda é pequeno, é possível contratar um plano mais barato, usar recursos mais abrangentes quando precisar e depois voltar a usar o plano básico. Justamente por isso os serviços em nuvem são vistos como os mais flexíveis;
  • Não é necessário implantação robusta de hardware necessitando apenas de um bom link de internet;
  • Fácil implantação, os serviços requeridos para execução podem estar disponíveis no ato da contratação não necessitando implantá-lo apenas configurá-lo;
  • Suporte.

Requisitos

Conforme testes realizados com a aplicação desenvolvida, a hospedagem a ser contratada deve atender os seguintes requisitos:

  • 512MB de Memória RAM;
  • 2GB de armazenamento;
  • Uma conta de e-mail para envio mensagens dos pedidos efetuados no site pelos clientes;
  • Base de dados flexível devido ao uso do hibernate;
  • Sistema operacional Linux ou Windows;
  • Sistema operacional Linux ou Windows;
  • Link de internet de 10Mbps.

Hospedagem

Na pesquisa por hospedagem online encontramos o site: https://uolhost.uol.com.br/hospedagem-de-sites/linguagem-programacao/java.html, este host atende todos os requisitos necessários por um preço acessível.
